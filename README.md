# Rice-Crop-Disease-Pest-Detection-using-YOLO
Object Detection Using YOLO.
This project uses YOLO (You Only Look Once) object detection to identify and localize rice crop diseases and pests from images.

The model predicts:

Object class (type of disease/pest)
Bounding box (location in the image)

Dataset contains 24 classes including pests like:

- RiceLeafRoller
- BrownPlantHopper 
- RiceBlast 
- RiceHispa 
- RiceFalseSmut, etc.

Tech Stack 
- Python 
- Ultralytics YOLO (YOLOv11n)
- NumPy, Pandas, Scikit-learn
 
Dataset

- Train images 
- Test images 
- Bounding box annotations (train.csv)
